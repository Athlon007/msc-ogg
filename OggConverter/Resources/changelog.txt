What's new in this release?

- The tool is now called "My Summer Car Music Manager"
- Added music manager - you can now play, sort, move and delete songs right in the program
- You can now drag & drop music files to quickly convert - either on running app window, or on it's executable
- MSMM now can download and install updates automatically
- You can now press Shift + Check for update to force download latest version
- You can also use Shift + Open LOG folder to remove all old log files
- Same goes for Shift + Launch game - it lets you start the game with no Steam quicker
- Refactored whole code
- Removed NReco Video Converter - now the tool uses only FFmpeg
- Lots of bug fixes

For more visit https://gitlab.com/aathlon/msc-ogg
